# 介绍&好处

  • eslint 是一个语法规范检查的包，内置 280 多个规则。
  • 好处
    • 统一团队编码规范
    • 统一语法
    • 避免低级错误，减少 git 不必要的提交
    • 在编译时检查语法，而不是在运行时检查

# Eslint 包

  • 安装方式
    • 通过 npm 进行全局安装或本地安装（项目）npm i eslint -D
    • 项目创建时选择 eslint 包
