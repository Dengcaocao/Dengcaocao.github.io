# 添加一个插件

  1. 下载插件
  2. 在vite.config.js配置文件中的plugins数组中进行配置。
plugins: [
  legacy({
    targets: ["defaults", "not IE 11"],
  }),
];

# 强制插件排序

为了更好的兼容 Rollup 插件，可通过enforce修饰符来进行插件排序。

  • pre：在 Vite 核心插件之前调用该插件
  • 默认：在 Vite 核心插件之后调用该插件
  • post：在 Vite 构建插件之后调用该插件
import image from "@rollup/plugin-image";
import { defineConfig } from "vite";

export default defineConfig({
  plugins: [
    {
      ...image(),
      enforce: "pre",
    },
  ],
});

# 按需应用

默认情况下，插件在开放和生产中都会应用。可通过修饰符apply来指定应用的环境。

  • serve 开发
  • build 生产
plugins: [
  {
    ...typescript2(),
    apply: "build",
  },
];
