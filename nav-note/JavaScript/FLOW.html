# 语句块

{
  statement_1
  statement_2
  statement_3
  ...
  statement_n
}

# if...else

if (condition) {
  statement_1
} else {
  statement_2
} // 语句 else 可选

if (condition_1) {
  statement_1

  statement_2
}else if (condition_n_1) {
  statement_n

  statement_last

# switch 语句

对表达式（expression）进行求值，并进行 case 语句匹配，匹配成功执行对应语句。未匹配到执行 default 语句。

default break 是可选的。break 是用于语句执行完成后跳出 swicth 语句。否则会继续执行后面的语句。

switch (expression) {
  case label_1:
  statements_1
  [break;]
  case label_2:
  statements_2
  [break;]
   ...
  default:
  statements_def
  [break;]
}

# 循环

# for

用于次数确定的循环。

for ([initialExpression]; [condition]; [incrementExpression]) {
  statement
}
1
2
3
  1. initialExpression 初始化表达式的值，可以声明变量。必须的。
  2. condition 计算表达式的值（true/false）。如果 condition 表达式整个都被省略掉了，condition 的值会被认为是 true。
  3. incrementExpression 更新表达式。可选。
  4. continue 可结束本次循环。

例子

for (let i = 0; i < 3; i++) {
  console.log(i)
}
1
2
3

continue 和 break 的区别

  1. continue：for 循环内部关键字。跳过本次循环，继续执行后面的循环。
  2. break：结束循环，后面的循环不再执行。只会跳出最近的这层循环，也可跳出指定循环。
// 跳出指定循环
loop1: for (let i = 0; i < 3; i++) {
  console.log(i, 'i')
  loop2: for (let j = 0; j < 3; j++) {
    if (j > 1) break loop1
    console.log(j, 'j')
  }
}
