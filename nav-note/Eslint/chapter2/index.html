# 生成 Eslint 配置文件

eslint 可以创建独立的配置文件.eslintrc.js，也可以在 package 中配置

  • 初始化 eslint 配置文件
1. /node_modules/.bin/eslint --init
2. npx eslint --init

    4. 是否使用 TS
    5. 规范之后的代码在哪里运行
    6. 选择配置规则
    7. 标准的那一套
    8. 文件使用语言

# 生成 Eslint 配置文件报错

  • 在后面检查代码规范时，会导致检查失败：Invalid ecmaVersion
  • 原因：创建文件第一步时选择了强制应用规范，会降低 eslint 版本
  • 解决方案：修改配置文件，降低 ecmaVersion, 从 13 => 12
