Object3D 是 Three.js 中大部分对象的基类，提供了一系列的属性和方法来对三维空间中的物体进行操纵。

注意

可以通过 .add( object ) 方法来将对象进行组合，该方法将对象添加为子对象，但为此最好使用Group（来作为父对象）。

# 属性

# .position

  • 表示物体位置的三维向量(Vector3)对象。默认值为(0, 0, 0)。
  • 可通过Vector3下的 set 方法设置x、y、z的位置。也可以通过 Vector3 下的属性 x、y、z 单独进行设置。
cube.position.set(1, 1, 1)
// 单独设置
cube.position.x = 3

# .scale

  • 控制物体缩放的三维向量(Vector3)对象。默认值是( 1, 1, 1 )。
  • 可通过Vector3下的 set 方法设置x、y、z的缩放。也可以通过 Vector3 下的属性 x、y、z 单独进行设置。
cube.scale.set(2, 1, 1)
// 单独设置
cube.scale.x = 3

3

# .rotation

  • 控制物体的旋转 Euler（欧拉角）对象，以弧度（2*PI）来表示。
  • 可通过Euler对象下的 set 方法设置x、y、z的旋转角度。也可以通过 Euler 下的属性 x、y、z 单独进行设置。
cube.rotation.set(Math.PI / 4, Math.PI / 4, Math.PI / 4)
// 单独设置
cube.rotation.x = PI / 2
