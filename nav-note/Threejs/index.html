# 摄像机

# 透视相机（PerspectiveCamera）。

  • 参数一：视野角度。在显示器上看到的场景范围。
  • 参数二：长宽比。
  • 参数三、四：近截面、远截面。当物体的某些部分比相机近截面近或远截面远的时候，这些部分将不会渲染到场景中。

# 几何体

# 立方缓冲几何体(BoxGeometry)

BoxGeometry 是四边形的原始几何类。通过构造函数提供的 Widht、Height、depth 参数来创建立方体或不规则四边形。

const geometry = new BoxGeometry(width, height, depth);

// example
const geometry = new THREE.BoxGeometry(1, 1, 1);

# 材质

# 基础网格材质(MeshBasicMaterial)

简单着色（平面或线框）方式来绘制几何体的材质。

# 构造函数

MeshBasicMaterial(parameters: object)

// example
const material = new THREE.MeshBasicMaterial({ color: 0xffffff })
1
2
3
4
  • parameters - 定义材质外观的对象。
  • 属性color可作为十六进制的字符串传递，默认 0xffffff，内部调用 Color.set(color)。

# 物体

# 网格(Mesh)

创建以三角形为 polygon mesh（多边形网格）的物体的类。

# 构造器

Mesh(geometry: BufferGeometry, material: Material | Array<Material>)

// example
const geometry = new THREE.BoxGeometry( 1, 1, 1 );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const mesh = new THREE.Mesh( geometry, material );

  • geometry - 几何体实例。
  • material - 材质或一个包含材质的数组。

# 控制

# 轨道控制器(OrbitControls)

可以使得相机围绕目标进行轨道运动。 需从examples/jsm/controls/OrbitControls.js文件中获取OrbitControls构造函数。

  • 参数一：需要的相控制机。
  • 参数二：渲染器的DOM元素。
import {OrbitControls} from 'three/examples/jsm/controls/OrbitControls'
const OrbitControls = new OrbitControls(camera, renderer.domElement)
1
2

# 属性

  • .enableDamping：是否开启阻尼（惯性），这将给控制器带来重量感。

注意

该值设置为 true 后，必须在动画循环里调用.update()。

# 方法

  • .update()：更新控制器。必须在摄像机的变换发生任何手动改变后调用。

# 辅助对象

# AxesHelper

简单模拟3个坐标轴的对象，红色代表 X 轴. 绿色代表 Y 轴. 蓝色代表 Z 轴。

const axesHelper = new THREE.AxesHelper(size: number);
scene.add( axesHelper );
1
2
  • size：轴的线段长度， 默认为 1。
